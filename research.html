Research

Navigation Bar

The inspiration for the navigation bar was taken from the navigation bar of Apple's Website. The contrast between a dark background and a lighter text looked great to take inspiration from.
This website uses a more simplistic approch to Apple's Navigation Bar. Since there is not as much content as in Apple's website, the navigation bar contains less buttons.

Home Page

The inspiration for the home page was taken from the home page of Spotify's Website. The minimal content with buttons to navigate through the website looked intuitive.
This website uses a more plain approach to Spotify's Home Page. This website also uses less bright colors as per the preference of the designer to look clean, simple and effective.

Blog Page

The inspiration for the Blog Page was taken from most blog pages but mainly from the Blog Page of Benedict Evans. The simplistic content layout with a clean looked very attractive.
The website uses a similar approach to Evan's Blog Page. The difference would be the use of image slider in this Blog Page and a simple photo in Evan's.

Contact Us Page

The inspiration for the Contact Us page was taken from the Contact Us page of Herald College. The layout is quite simple and easy to understand.
This website uses a similar approch to Herald's Website. It also requires the user to enter data otherwise the data will not go through.

Footer

The inspiration for the Footer was taken from the website of Youtuber Binging with Babish. The layout is clean, simple and minimalistic.
This website also uses a similar approach to the website by keeping the footer simple and minimal containing information about the social media and copyright information.
